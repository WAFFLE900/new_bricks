# 歡迎來到故事書 ( Storybook )

Storybook 可以讓您獨立地構建 UI 組件，而不需要與應用程序的業務邏輯、數據和上下文進行交互，從而輕鬆開發難以達成的狀態。將這些 UI 狀態保存為**stories**，以便在開發、測試或 QA 期間重新查看。

現在可以通過在側欄中導航到它們來瀏覽示例故事。在`stories`目錄中查看它們的代碼，以了解它們的工作原理。我們建議使用[**組件驅動**](https://componentdriven.org)的流程來構建 UI，從原子組件開始，最終到頁面。
